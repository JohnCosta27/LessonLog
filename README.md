# JohnTech

A collection of projects which allows my projects to run, this repo is used as a mono-repo, which does mean I need to manage it much better than any other repo.

The microservices will consist of a docker container each, and nginx acting as a gateway between all of them. Each one should have its own database, which will be more of a challenge, but definetly achievable.

## Projects

### Auth

This is the micro service which provides authentication and authorisation systems, it allows people to login and register across my projects.
